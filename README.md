\# GenAI Proof of Concepts (PoCs)



📌 Overview

This repository contains a collection of \*\*Generative AI-based Proof of Concepts (PoCs)\*\* developed to explore different problem-solving approaches using modern AI techniques.



Unlike conventional methods, these projects focus on:

\- Experimentation

\- Rapid prototyping

\- Creative problem-solving using GenAI



\---



💡 Philosophy

I believe in \*\*not always following a traditional or conventional path\*\* to solve problems.



This repository reflects that mindset.



Each PoC here is built with the intent to:

\- Explore alternative approaches

\- Leverage Generative AI capabilities

\- Test ideas quickly and iteratively



\---



🚀 What You’ll Find Here

This repo includes multiple experiments such as:

\- Graph-based RAG (Retrieval Augmented Generation)

\- Named Entity Recognition (NER) models

\- OCR + KPI extraction modules

\- LLM-based transformations and workflows

\- Observability tools (e.g., Langfuse, MLflow)



\---



⚙️ Tech Stack

\- Python

\- OpenAI / LLM APIs

\- LangChain / GraphRAG

\- NLP (spaCy, custom pipelines)

\- OCR (Tesseract)

\- MLflow / Langfuse
